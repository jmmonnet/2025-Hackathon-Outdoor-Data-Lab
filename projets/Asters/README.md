## Description des fichiers
### strava.R
Script R de traitement des données Strava Metro (compte de connexion ASTERS)
"Run,Walk,Hike" et "Ride,EBikeRide"
Du 01/06/2024 au 31/08/2024

### tourbieres.shp
Contours des tourbières à enjeux présentes dans la réserve des Contamines-Montjoie

### ride_problemes.shp
Intersection spatiale entre les traces de vélos issues de Strava et les tourbières à enjeux

### skitour.geojson
Données GPX issues du site web Skitour dans le Beaufortain et le Massif du Mont Blanc
