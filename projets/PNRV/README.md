# Groupe Vercors

## Cas d'dusage

Un gestionnaire ouvre une carte. Pour chaque maille, un indice de risque (enjeu * frequenation * pratique)
sur chaque maille une couleur  (diagramme de Venn) associées aà l'indice de risque, ainsi une info qui liste les enjeux, la frequentation (sur une temporalité) et les pratiques outdoor de la maille. 
A partir de la maille on peut générer un rapport

## Données sources

- Observation sensibles -> GenoNature PNRV API Export
- Indice de fréquentation -> OutdoorVision Plateforme .
- Aire protégées -> Flux WFS Carmen ?
- Zones sensibles publiées -> Biodivsport
- Pratique outdoor -> GeoTrek PNRV API Export

## Objectif Hackatoon 

- Données statiques (import des données plus tard via FLUX)
- Lancer le traitement QGIS pour générer une carte (raster)
- Création de rapport a partir de la couche
